# Tech Stack & Tools

- **Core Engine:** GitHub Actions (cron: `0 0 * * *`).
- **Stats Generation:** `lowlighter/metrics` for advanced infographics.
- **Language/Rank Cards:** `anuraghazra/github-readme-stats` (Self-hosted on Vercel recommended).
- **Layout:** Sanitized HTML `<table>` tags for multi-column grids.
- **Visuals:** `readme-typing-svg` for headers and `github-profile-views-counter`.

## Naming Conventions

- **Assets:** Store all generated SVGs in `assets/`.
- **Workflows:** Use descriptive names (e.g., `metrics-update.yml`).
